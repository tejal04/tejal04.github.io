Call Selling Price Prediction

An end-to-end Machine Learning project that helps users to get an estimated selling price for their used car based on various features, which can help in making informed decisions when selling a car.

Data :
Data is collected from Car Dekho portal (a website to buy new/used cars in India).

Machine Learning :
Performed well thorough EDA including data cleaning, pre-processing, analysis to identify trends and patterns in the data, feature engineering, partitioning and model selection. Since the model predicts the price, it is a regression task and the Random Forest regressor proved to have best performance.

Deployment :
The trained model was deployed using Flask web application. The front end interface was developed using HTML, CSS, Javascript.

Tech Stack :
python, pandas, numpy, seaborn, mathplotlib, scikit-learn, flask, API.

Movie Review Analysis

Customer centric businesses heavily rely on product reviews to improve their‚Äã product and maximize their market value. In this project we focus to perform sentiment analysis on the reviews to gauge the overall positive or negative valence of the review and then perform topic modeling to see what topics are prominent from the reviews for a particular movie.

Data :
1. IMDb 50K movie reviews dataset released by Stanford.
2. Reviews of The Shawshank Redemption scraped from IMDb using Selenium.

Sentiment Analysis :
Transformed text into BOW and TFIDF vector representations.
Modeled three machine learning models - Naive Bayes, Decision Tree and Random Forest to predict the sentiment.
Also, developed Deep Learning models like LSTM and RNN.
Performed comparative analysis to find the best model performance.

Topic Modelling :
1. LDA with BOW and TFIDF vector representations.
2. NMF with BOW and TFIDF vector representations.

Tech Stack :
python, pandas, numpy, seaborn, mathplotlib, scikit-learn, gensim, nltk, pytorch.

NYC TLC DATA ENGINEERING

This project focus on NYC TLC Data Analysis using Python, GCP Storage, Compute Engine, Mage Data Pipeline Tool, BigQuery, and Looker Studio. Aims to extract insights from the dataset for informed decisions and deeper operational understanding.

Data :
Data is provided by NYC Taxi and Limousine Commission (TLC) on their portal.

Data Engineering :
1. Modeled data using star schema.
2. Performed ETL (Extract-Transform-Load) on the data with the help of Mage Pipeline.
3. Used BigQuery to query the loaded data.
4. Visualized various trends in the data using Looker Studio.

Tech Stack :
python, pandas, numpy, seaborn, mathplotlib, Google Cloud : GCP Storage, Compute Engine, BigQuery, Mage Data Pipeline, Google Looker Studio.
